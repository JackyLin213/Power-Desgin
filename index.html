EA 電源補償器互動解析

1. 簡介與實體架構

Type 1 補償器 (積分器)：
僅包含一個原點極點，提供極高的直流增益以消除穩態誤差。通常用於頻寬要求不高，或系統受控體（Plant）本身只具有單極點特性的場合。
Type 2 補償器：
包含 1個原點極點、1個零點 與 1個高頻極點。最常用於「電流模式控制」的轉換器，利用其零點補償負載極點，並用高頻極點抵消 ESR 零點。
Type 3 補償器：
具有 2個零點 和 3個極點，能提供高達 180 度的相位提升，專門用於補償具有 LC 雙極點特性的「電壓模式控制」轉換器。
✨ 完整實體電路圖 (含分壓電阻與參考電壓)

2. 轉移函數與推導

💡 關於分壓電阻 (\(R_{bot}\)) 與參考電壓 (\(V_{REF}\))：
在實際電路中，Op-Amp 的 (+) 端接有直流參考電壓 \(V_{REF}\)，(-) 端節點為 \(V_{FB}\)，並透過 \(R_{bot}\) 接地以設定直流輸出電壓。
但在進行 AC 小信號分析時，直流量 \(V_{REF}\) 視為 AC 接地。根據虛接地 (Virtual Ground) 原理，\(V_{FB}\) 的 AC 小信號也為 0。因此，沒有 AC 訊號電流流過 \(R_{bot}\)，所以 \(R_{bot}\) 並不會出現在 AC 轉移函數 \(H(s)\) 之中。

步驟 1：阻抗計算

輸入阻抗：\(Z_{in}(s) = R_1\)

回授阻抗：\(Z_f(s) = \frac{1}{sC_1}\)

步驟 2：組合轉移函數

$$H(s) = -\frac{Z_f(s)}{Z_{in}(s)} = -\frac{\frac{1}{sC_1}}{R_1} = -\frac{1}{s R_1 C_1}$$

步驟 1：阻抗計算

輸入阻抗：\(Z_{in}(s) = R_1\)

回授阻抗 \(Z_f(s)\) 是 \(C_3\) 與 \((R_2 + \frac{1}{sC_2})\) 的並聯。原始公式為：

$$Z_f(s) = \left( R_2 + \frac{1}{sC_2} \right) \parallel \left( \frac{1}{sC_3} \right) = \frac{\left( R_2 + \frac{1}{sC_2} \right) \cdot \left( \frac{1}{sC_3} \right)}{\left( R_2 + \frac{1}{sC_2} \right) + \frac{1}{sC_3}}$$

將分子與分母同乘 \(s^2 C_2 C_3\) 進行通分化簡：

$$Z_f(s) = \frac{s R_2 C_2 + 1}{s^2 R_2 C_2 C_3 + s(C_2 + C_3)}$$

分母提出 \(s(C_2 + C_3)\) 整理成標準極點形式：

$$Z_f(s) = \frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}$$

步驟 2：組合轉移函數

代入 \(H(s) = -\frac{Z_f(s)}{Z_{in}(s)}\) 並整理（為分析頻率響應，後續忽略反相負號）：

$$H(s) = \frac{\frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}}{R_1}$$

$$H(s) = \frac{1 + s R_2 C_2}{s R_1 (C_2 + C_3) \left[ 1 + s R_2 \left(\frac{C_2 C_3}{C_2 + C_3}\right) \right]}$$

步驟 1：輸入阻抗 \(Z_{in}(s)\)

\(Z_{in}\) 是 \(R_1\) 與 \((R_3 + \frac{1}{sC_1})\) 的並聯。原始公式為：

$$Z_{in}(s) = R_1 \parallel \left( R_3 + \frac{1}{sC_1} \right) = \frac{R_1 \cdot \left( R_3 + \frac{1}{sC_1} \right)}{R_1 + \left( R_3 + \frac{1}{sC_1} \right)}$$

將分子與分母同乘 \(s C_1\) 化簡以消去分母的分數：

$$Z_{in}(s) = \frac{R_1 (s R_3 C_1 + 1)}{s R_1 C_1 + s R_3 C_1 + 1} = \frac{R_1 (s R_3 C_1 + 1)}{s (R_1 + R_3) C_1 + 1}$$

步驟 2：回授阻抗 \(Z_f(s)\)

\(Z_f\) 是 \(C_3\) 與 \((R_2 + \frac{1}{sC_2})\) 的並聯。原始公式與化簡（同 Type 2）：

$$Z_f(s) = \left( R_2 + \frac{1}{sC_2} \right) \parallel \left( \frac{1}{sC_3} \right) = \frac{\left( R_2 + \frac{1}{sC_2} \right) \cdot \left( \frac{1}{sC_3} \right)}{\left( R_2 + \frac{1}{sC_2} \right) + \frac{1}{sC_3}}$$

$$Z_f(s) = \frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}$$

步驟 3：組合轉移函數

代入 \(H(s) = -\frac{Z_f(s)}{Z_{in}(s)}\)（為分析頻率響應，忽略負號）：

$$H(s) = \frac{\frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}}{\frac{R_1 (s R_3 C_1 + 1)}{s (R_1 + R_3) C_1 + 1}}$$

將繁分數展開整理，分離常數、分子（零點）與分母（極點）：

$$H(s) = \frac{1}{s R_1 (C_2 + C_3)} \cdot \frac{(s R_2 C_2 + 1) \left[ s (R_1 + R_3) C_1 + 1 \right]}{(s R_3 C_1 + 1) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}$$

標準形式極零點分析：

$$H(s) = \frac{1}{s R_1 C_1} = \frac{\omega_{p0}}{s}$$

原點極點 (fp0) \( f_{p0} = \frac{1}{2\pi R_1 C_1} \)

標準形式極零點分析：

$$H(s) = \frac{\omega_{p0}}{s} \cdot \frac{1 + \frac{s}{\omega_{z1}}}{1 + \frac{s}{\omega_{p1}}}$$

原點極點 (fp0) \( f_{p0} = \frac{1}{2\pi R_1 (C_2 + C_3)} \)
零點 (fz1) \( f_{z1} = \frac{1}{2\pi R_2 C_2} \)
極點 (fp1) \( f_{p1} = \frac{C_2 + C_3}{2\pi R_2 C_2 C_3} \approx \frac{1}{2\pi R_2 C_3} \)

標準形式極零點分析：

$$H(s) = \frac{\omega_{p0}}{s} \cdot \frac{(1 + \frac{s}{\omega_{z1}})(1 + \frac{s}{\omega_{z2}})}{(1 + \frac{s}{\omega_{p1}})(1 + \frac{s}{\omega_{p2}})}$$

原點極點 (fp0) \( f_{p0} = \frac{1}{2\pi R_1 (C_2 + C_3)} \)
第一零點 (fz1) \( f_{z1} = \frac{1}{2\pi R_2 C_2} \)
第二零點 (fz2) \( f_{z2} = \frac{1}{2\pi (R_1 + R_3) C_1} \)
第一極點 (fp1) \( f_{p1} = \frac{1}{2\pi R_3 C_1} \)
第二極點 (fp2) \( f_{p2} = \frac{C_2 + C_3}{2\pi R_2 C_2 C_3} \approx \frac{1}{2\pi R_2 C_3} \)
