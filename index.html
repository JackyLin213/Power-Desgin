2. 轉移函數分析

💡 關於分壓電阻 (\(R_{bot}\)) 與參考電壓 (\(V_{REF}\))：
AC 分析時，直流 \(V_{REF}\) 視為 AC 接地。根據虛接地原理，\(V_{FB}\) 的 AC 小信號也為 0。因此，無 AC 訊號電流流過 \(R_{bot}\)，\(R_{bot}\) 並不會出現在轉移函數 \(H(s)\) 中。

步驟 1：阻抗計算

輸入阻抗：\(Z_{in}(s) = R_1\)，回授阻抗：\(Z_f(s) = \frac{1}{sC_1}\)

步驟 2：組合轉移函數

$$H(s) = -\frac{Z_f(s)}{Z_{in}(s)} = -\frac{1}{s R_1 C_1}$$

步驟 1：阻抗計算

輸入阻抗：\(Z_{in}(s) = R_1\)

\(Z_f(s) = \left( R_2 + \frac{1}{sC_2} \right) \parallel \left( \frac{1}{sC_3} \right) = \frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}\)

步驟 2：組合轉移函數 (忽略負號)

$$H(s) = \frac{1 + s R_2 C_2}{s R_1 (C_2 + C_3) \left[ 1 + s R_2 \left(\frac{C_2 C_3}{C_2 + C_3}\right) \right]}$$

步驟 1：輸入阻抗 \(Z_{in}(s)\)

$$Z_{in}(s) = R_1 \parallel \left( R_3 + \frac{1}{sC_1} \right) = \frac{R_1 (s R_3 C_1 + 1)}{s (R_1 + R_3) C_1 + 1}$$

步驟 2：回授阻抗 \(Z_f(s)\)

$$Z_f(s) = \left( R_2 + \frac{1}{sC_2} \right) \parallel \left( \frac{1}{sC_3} \right) = \frac{s R_2 C_2 + 1}{s (C_2 + C_3) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}$$

步驟 3：組合轉移函數

$$H(s) = \frac{1}{s R_1 (C_2 + C_3)} \cdot \frac{(s R_2 C_2 + 1) \left[ s (R_1 + R_3) C_1 + 1 \right]}{(s R_3 C_1 + 1) \left[ s \frac{R_2 C_2 C_3}{C_2 + C_3} + 1 \right]}$$
